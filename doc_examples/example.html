Replot Map: Examples

Basic Example

In this example, we rely completely on Replot's smart defaults - simply plug in the data and select the country - and allow Replot to do the rest. We get a beautiful interactive map with just a few lines of code.

Custom Dimensions

The map can be displayed in custom dimensions using the width prop. Height will be adjusted automatically to fit the entirity of map without skewing the original aspect ratio.

Custom Colors

The map can be colored with a custom range of gradient using the colorRangeLow and colorRangeHigh props.

The color scale can be switched between linear and logarithmic using the scale prop.

Zoom Features

The map can be centered on a specific area by some zoom scale using the zoomID and zoomScale props.

When zoomScale is specified, but zoomID is not, the map will be scaled up from its top left corner by zoomScale.

Tooltip Features

The tooltip color can be switched between light and dark using the tooltipColor prop.

The tooltip can be switched on or off using the tooltip prop.

The tooltip can display custom contents using the tooltipContent prop.
